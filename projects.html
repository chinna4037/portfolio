Incremental backup using flutter

Incremental backup is a method of backing up data where only the changes made since the last backup are stored. This approach helps reduce storage space and backup time compared to full backups, which involve copying all data each time. Unlike full backups which copy everything, incremental backups focus solely on changes since the last backup. It's like keeping a running log of modifications, ensuring you capture the latest revisions without replicating data already preserved.
